ZENSEA <br/>

<p>Zensea is a full-stack web platform built for scuba diving enthusiasts. It provides an intuitive interface to explore scuba diving destinations, course information, and engagement features in one place.

Features <br/>
Responsive and user-friendly UI<br/>
Scuba diving destination and course details<br/>
Backend APIs for dynamic data handling<br/>
Scalable multi-tier architecture<br/>

Tech Stack <br/>
Frontend: HTML, CSS, JavaScript, React.js<br/>
Backend: Node.js, Express.js<br/>
Database: MongoDB<br/>

<i><b>*This project was developed as a learning-based full-stack application to gain hands-on experience with modern web technologies and real-world architecture.*<b/><i/>
